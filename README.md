# Superstore Sales Performance Dashboard (Excel Project)

# Project Overview

This project presents an interactive Sales Performance Dashboard built using Microsoft Excel (2007) based on the popular Sample Superstore dataset. The dashboard provides key insights into sales, profit, customer segments, and product performance to support business decision-making.


## Dataset Description

The dataset includes transactional-level data with the following fields:

- Ship Mode
- Segment
- Country, City, State, Postal Code, Region
- Category & Sub-Category
- Sales, Quantity, Discount, Profit


## Key KPIs

-  Total Sales: $2.30M
-  Total Profit: $0.29M
-  Profit Margin: 12%
-  Total Orders: 9,994


## Dashboard Features

 ### 1. Regional Performance Analysis

- Displays total sales, profit, and order volume by region
- Helps understand profitable region

### 2. Category Performance

- Compares Sales vs Profit across:
  - Technology
  - Furniture
  - Office Supplies
- Identifies most profitable product categories

### 3. Customer Segment Analysis

- Analyzes performance across:
  - Consumer
  - Corporate
  - Home Office
- Highlights key revenue-driving segments

### 4. Discount vs Profit Analysis

- Shows impact of discount levels on profitability
- Highlights negative correlation between discount and profit
  
### 5. Sub-Category Analysis

- Top 5 Sub-Categories by Profit
  - Copiers, Phones, Accessories, Paper, Binders
- Bottom 5 Sub-Categories by Profit
  - Tables (loss), Bookcases, Supplies, etc.


## *Key Insights*

- Technology is the most profitable category
- Consumer segment contributes the highest sales and profit
- High discounts significantly reduce profitability
- Certain sub-categories like Tables generate losses
- Copiers and Phones are major profit drivers


 ## Tools & Techniques Used

- Microsoft Excel 2007
- Pivot Tables
- Charts & Graphs
- KPI Cards
- Data Aggregation & Analysis


## Limitations

- Built using Excel 2007 (limited visualization features)
- No dynamic slicers or advanced interactivity
- Limited time-series analysis
